Banco Fichin - Cajero automático

Descripción del ejercicio

Este ejercicio simula un cajero automático básico utilizando JavaScript puro. El sistema solicita al usuario que ingrese su nombre de usuario y contraseña mediante cuadros de diálogo prompt(), y valida esos datos contra una lista predefinida.

Si las credenciales son correctas, permite realizar un retiro de dinero, verificando que haya fondos disponibles. Las acciones y mensajes se muestran mediante alert() y la información del back para testeo se hace mediante console.log().

Recursos de JavaScript utilizados

  • Variables (let, const)
  • Arrays y métodos como .indexOf()
  • Bucles for y while
  • Condicionales if, else y switch
  • Funciones flecha
  • Conversión de tipos con Number() y validación con isNaN()
  • Interacción con el usuario: alert(), prompt(), console.log()
